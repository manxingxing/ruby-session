# Ruby 介绍
![Matz](./images/matz.png)

* 脚本语言，类似 python, js
* 彻底的面向对象的语言: 一切皆对象(没有原生类型和对象的区分)
* 单继承
* 没有 interface, 没有范型
* module(mixin)
* duck-typing
* 强大(到失控)的元编程: 可以在运行时添加/修改/删除 类、方法
